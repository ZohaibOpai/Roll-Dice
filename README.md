## 📌 Overview

Dice Forge is a modern, interactive, and visually stunning dice rolling web application built using **HTML, CSS, and JavaScript**. It features advanced animations, multiple game modes, and real-time statistics tracking.

---

## ✨ Features

### 🎯 Core Features

* Roll 1 to 6 dice dynamically
* 3D animated dice with realistic motion
* Smooth UI with neon futuristic design
* Total score calculation

### 🎮 Game Modes

* **Free Mode** – Unlimited rolling
* **1v1 Battle Mode** – Compete against another player
* **Target Mode** – Reach exact score to win

### 📊 Statistics

* Total rolls tracking
* Average score
* Highest & lowest rolls
* Count of sixes and ones
* Frequency distribution chart

### 📜 History

* Last 30 rolls stored
* Visual chips for each dice value

### 🎨 UI/UX

* Fully responsive design
* Animated background effects
* Confetti and particle effects
* Interactive hover & ripple effects
